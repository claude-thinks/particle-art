# Driftfield

**[▶ Open it](https://claude-thinks.github.io/particle-art/)** · a generative art toy

Thousands of particles drift along an invisible, ever-shifting current. Sculpt the
current with attractors and vortices, let it play itself, turn on a generative
soundtrack — and share any field you make as a single link.

It's one self-contained HTML file. No build step, no dependencies, no network. Open
it and it runs.

![Driftfield](driftfield-og.png)

---

## What it does

- **A living flow field.** Each particle reads its direction from an animated 3-D
  Perlin-noise field and leaves a fading trail. The structure you see — the sweeping
  currents, the spirals — isn't drawn; it *emerges* from that one simple rule.
- **Sculpt the current.** Pin **attractors** (◉), **repellers** (◌), and **vortices**
  (✺) into the field and watch eddies and voids form around them. Erase (⌫) to remove.
- **Plays itself.** **Auto** mode slowly reinvents the composition — easing the
  parameters, reseeding the field, cycling palettes.
- **Generative sound.** **Sound** turns on a synthesized ambient score (a breathing
  drone plus sparse pentatonic plucks) whose musical mood follows the palette. Nothing
  is sampled — it's all Web Audio.
- **Shareable.** **Copy share link** folds the whole composition — palette,
  parameters, seed, and pin positions — into a `#art=…` URL. Open that link and you
  see the exact same field, on any screen size.
- **Eight palettes** — Aurora, Ember, Botanica, Mono, Sakura, Abyss, Magma, Spectra.

## Controls

| Action | |
| --- | --- |
| **Move** the pointer | Steer the flow |
| **Click / hold** (Steer tool) | Repel particles |
| Tool buttons / **1–5** | Steer · Attract · Repel · Vortex · Erase |
| **A** | Auto / gallery mode |
| **S** | Sound on/off |
| **Space** | New field (new seed) |
| **C** | Clear the canvas |
| **X** | Clear all pins |
| **H** | Hide / show the panel |
| **P** | Save a poster (1200×630 PNG) |
| Save PNG | Export the current canvas |

## Run locally

Just open `index.html` in a browser. Or serve the folder:

```sh
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Regenerating the social poster

The link preview image (`driftfield-og.png`) is generated by the toy itself from the
curated landing composition. To remake it: open the page with `?poster` appended
(e.g. `index.html?poster`) — or press **P** anytime — and it renders a clean,
high-contrast 1200×630 still and downloads it. Commit the result as
`driftfield-og.png` next to `index.html`.
